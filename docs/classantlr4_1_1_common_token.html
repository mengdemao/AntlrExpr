◆ CommonToken() [4/4]

CommonToken::CommonToken ( Token *  oldToken)

Constructs a new CommonToken as a copy of another Token.

If

oldToken

is also a CommonToken instance, the newly constructed token will share a reference to the text field and the Pair stored in source. Otherwise, text will be assigned the result of calling getText, and source will be constructed from the result of Token#getTokenSource and Token#getInputStream.

参数
oldTokenThe token to copy.

size_t CommonToken::getTokenIndex ( ) const
overridevirtual

An index from 0..n-1 of the token object in the input stream. This must be valid in order to print token streams and use TokenRewriteStream.

Return INVALID_INDEX to indicate that this token was conjured up since it doesn't have a valid index.

std::pair<TokenSource *, CharStream *> antlr4::CommonToken::_source
protected

This is the backing field for getTokenSource and getInputStream.

These properties share a field to reduce the memory footprint of CommonToken. Tokens created by a CommonTokenFactory from the same source and input stream share a reference to the same Pair containing these values.
